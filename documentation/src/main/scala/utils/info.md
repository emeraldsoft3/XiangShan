# utils

This folder contains general-purpose utilities and helper classes used throughout the XiangShan codebase.

## Overview
These are not specific to the processor architecture but provide common hardware description patterns or Scala language extensions.

## Key Utilities
- **AXI4Lite.scala**: Helper for AXI4-Lite bus interfaces.
- **PipeWithFlush.scala**: A pipeline register that supports synchronous flushes.
- **MathUtils.scala / BitsUtils.scala**: Common mathematical and bit-manipulation functions.
- **TLDump.scala**: Tools for dumping TileLink traffic for debugging.
